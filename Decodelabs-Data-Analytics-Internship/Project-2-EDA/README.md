# Project 2 - Exploratory Data Analysis (EDA)

## Overview
This project involved performing exploratory data analysis on a transactional sales dataset to identify trends, patterns, distributions, and outliers.

- Descriptive statistics
- Trend analysis
- Distribution analysis
- Outlier detection
- Revenue analysis

- Matplotlib

- Data Cleaning and Preprocessing with Python
